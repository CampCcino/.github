##  캠핑의 온도를 담다, 캠프치노 (Camp Ccino) 
> 전국 캠핑장을 한눈에 탐색할 수 있는 캠핑장 통합 검색 웹사이트 캠프치노입니다.

<img width="627" height="347" alt="campccino" src="https://github.com/user-attachments/assets/56d4a442-497f-40d7-aae4-4eb2f05e0c4a" />


> Kakao Map API를 활용해 지역별 캠핑장의 위치를 직관적인 지도 형태로 제공하며, 리스트 뷰를 통해 주소, 상세 정보, 사용자 후기 등을 함께 확인할 수 있도록 구성하였습니다.
또한 게시판 기능을 통해 캠핑장 정보 공유와 사용자 간 의견 교환을 지원하고, 상점 기능을 연계하여 탐색부터 구매까지 이어지는 통합 사용자 경험을 제공하고자 하였습니다.
